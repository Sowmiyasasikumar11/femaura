# FemAura

A modern React + TypeScript wellness application for menstrual cycle care, mood tracking, guided chat support, awareness videos, and menopause planning.

## Features

- Secure login and signup user flows
- Personalized menstrual cycle tracker with calendar predictions
- Daily mood logging and trend insights
- Chatbot-style wellness guidance for menstruation and mood support
- Curated awareness videos for menstrual hygiene and product education
- Menopause support with symptom tracking and tailored wellbeing tips
- Responsive design using Tailwind CSS
- English/Tamil internationalization

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm

### Installation

```bash
npm install
```

### Running the Application

```bash
npm run dev
```

Open `http://localhost:5173` in your browser.

### Building for Production

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```

### Linting

```bash
npm run lint
```
